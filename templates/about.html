About the Project

The Movie Recommendation System is a web-based application designed to help users discover movies based on their preferences. The system analyzes user-selected movies and provides recommendations using the Apriori Algorithm, a popular technique in association rule mining.

How It Works

  1. Data Collection & Preprocessing: The system utilizes a dataset containing information about movies, including titles, genres, directors, cast, and IMDb ratings. The data is cleaned, formatted, and structured into transactions where each userâ€™s movie preferences are grouped together.
  2. Data Processing: The preprocessed movie data is converted into a format compatible with the Apriori Algorithm. Each movie is treated as an item, and sets of frequently watched movies are identified to find strong associations.
  3. Applying the Apriori Algorithm: The system uses the mlxtend library to apply the Apriori algorithm. It scans the dataset to find movies that are often watched together and generates association rules based on support, confidence, and lift.
  4. Recommendation Generation: When a user selects three movies, the system finds the strongest associations from the generated rules and suggests up to 10 relevant movies. The recommendations are ranked based on confidence scores to ensure the most meaningful suggestions.
  5. Fetching Movie Posters: To enhance the user experience, the system integrates the TMDB API to fetch and display high-quality movie posters. This makes the recommendations visually appealing and more engaging for users.
  6. User Interface: The frontend is designed with a clean and responsive UI, allowing users to interact easily with the recommendation system. Features include:
